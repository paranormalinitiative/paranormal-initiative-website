From A Question To A Platform

Several months ago I was asked a simple question: can you build an SLS system for Mac?

At first, the answer was not easy. Most Kinect-based SLS systems were designed around Microsoft's Kinect SDK and Windows hardware.

For investigators who primarily use Apple products, there simply were not many options available. Rather than forcing researchers to maintain a separate Windows machine just for Kinect mapping, I decided to see if it was possible to build something specifically for the Apple ecosystem.

That challenge became Aether Spectra SLS.

Built Around Real Sensor Data

What started as an experiment quickly evolved into a complete environmental mapping and paranormal experimentation platform built around real hardware, real sensor data, and full investigator transparency.

One thing that is extremely important to me is honesty.

This system is not using fake night vision filters. It is not using visual tricks designed to make ordinary footage appear more dramatic. It is not generating random evidence.

The RGB mode is real RGB camera data. The IR Night Vision mode is real Kinect infrared camera data. The depth mapping system is real Kinect depth information. The pose tracking system is built using Apple Vision analysis. The audio system captures real recorded audio.

Everything displayed on-screen originates from actual hardware sensors and documented processing.

Why Transparency Matters

As investigators, our reputation is everything.

I have spent years advocating for responsible investigation practices, proper documentation, and evidence-based research. I would never risk that reputation by building gimmicks, fake effects, or software designed to mislead people into believing something is happening when it is not.

Aether Spectra SLS was designed around one core principle: show investigators exactly what the hardware sees and document it as accurately as possible.

No mystery boxes. No hidden processing. No random word generators. No AI deciding what evidence means. No fake night vision filters. No gimmicks.

Just tools, documentation, experimentation, and investigation.

Current Feature Set

Current features include RGB Camera Mode, true Infrared Night Vision Mode, real-time Kinect depth mapping, Depth Heat Map visualization, Raw 11-Bit Depth View, Point Matrix visualization, and Apple Vision pose analysis.

The platform also includes adjustable evidence gate validation, Kinect motor tilt control, live EVP audio monitoring, voice-band waveform analysis, audio-only recording, audio and video recording, WAV forensic master audio capture, and MP4 session recording.

Automatic session documentation, timestamped session archives, multiple depth visualization palettes, and auto/manual recording FPS control are also part of the current workflow.

Designed For Investigators

Aether Spectra SLS is built for researchers. Built for experimenters. Built for investigators.

And most importantly, it is built for Mac users.

The goal is not to create theatrical footage or replace investigator judgment. The goal is to provide a transparent toolset for environmental mapping, structured experimentation, recording, documentation, and review.

Looking Ahead

Future iPhone and iPad versions are already planned and will bring portions of the Aether Spectra SLS platform to Apple's mobile ecosystem.
